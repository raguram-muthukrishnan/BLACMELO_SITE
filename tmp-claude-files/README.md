# Tmp Claude Files (Hydrogen Storefront)

This folder contains temporary files generated by Claude AI during development sessions for the hydrogen-storefront project.

These files are:
- Automatically ignored by git (see .gitignore)
- Safe to delete at any time
- Used for temporary storage during AI-assisted development

## Cleanup

You can safely delete all files in this folder:

```bash
# Windows
del /q hydrogen-storefront\tmp-claude-files\tmpclaude-*

# Linux/Mac
rm -f hydrogen-storefront/tmp-claude-files/tmpclaude-*
```
