## Summary

This visual guide shows how the color product architecture works:

1. **Products are grouped** by `color_family` metafield
2. **Loader fetches** related products via GraphQL
3. **ColorProductSwitcher** displays product swatches
4. **User clicks** color вҶ’ navigates to new product
5. **User selects** size вҶ’ changes variant within product
6. **Add to cart** вҶ’ correct product + variant added

The system provides a premium UX while maintaining clean separation between product-level (color) and variant-level (size) attributes.
